動詞マーカーと完了形の形

動詞マーカーの接頭辞

動詞マーカーはある種の語を特定します。もっともよく起きるのはmȩ（動詞が母音で始まる場合はm）です。そのほかにo-もあります。能格動詞ではmȩchat（mȩ + chat）「いぶされる」のように動詞マーカーがつきますが、非完了形ではmȩngat「いぶす」のようにより複雑です（mȩ + ng + chat）。以下に示すように多くの語にmȩがついています。

接中辞の動詞マーカーと音位転換

自動詞で接中辞の形の動詞マーカーを持つ動詞の場合、最初の子音の後に来ます。

他動詞では接中辞の動詞マーカーは-u-やまれに-o-として現れます。suebȩk 飛ぶ、ruebȩt 落ちる、soisȩb 入るなどです。いずれも唇音が入っており、異音としてmが使われなくなったと考えられます。

mȩ + sechȩrはs + mȩ + echȩrとなり、動詞マーカーのȩが消えます。これを音位転換といいます。動詞マーカーの音位転換はパラオ語の動詞の完了形で見られます。

mȩ + sebȩk （基本形） →
s + mȩ + ebȩk （音位転換） →
s + m + ebȩk （ȩの省略） →
s + u + ebȩk （異化）

動作動詞の過去形は過去形のマーカーである-il-か-ir-が使われます。これらには強勢は落ちません。

動詞仮定形での動詞マーカーの欠如

パラオ語の動詞の仮定形では動詞マーカーがなくなります。すなわち基本形の動詞マーカー＋非完了形マーカー＋動詞語幹の仮定形は仮定形代名詞＋非完了形マーカー＋動詞語幹となります。

上の形を分析すると以下の通りになります。

mȩ + bo （基本形） →

b + m + o (ȩの脱落) →
m + o (mの前のbの脱落)

冒頭でパラオ語の仮定形同氏は動詞マーカーを持たないといいましたが、いくつかの例外があります。一つは自動詞の状態動詞でmȩ-がつきます。mȩkngit「悪い」lȩmȩkngit「（もし）それが悪いなら」のように。また多くの自動詞の動作動詞も動詞マーカーがなくなりますが、ごく一部は保持します。また能格動詞の仮定形も動詞マーカーを保持します。

完了形における音位転換

3人称完了形の目的語代名詞は-iiか-ngiiまたは母音＋r（ir, urなど）です。一方3人称複数形の非人間代名詞は-øです。3人称単数の目的語代名詞はいつも強勢が落ちますが、動詞語幹は強勢が落ちません。一方、非人間の目的語代名詞は現れないため、完了形のduálȩm「それらを植える」などは動詞語幹に強勢が落ちます。

動詞の非完了形 動詞の完了形
動詞マーカー＋非完了形＋動詞語幹 動詞マーカー＋動詞語幹＋目的語代名詞

非完了形では非完了マーカー（-l-, -ng-, -m-）などのあとの動詞語幹の子音は脱落します。一方で完了形同士では保存されます。能格動詞でも動詞マーカー＋動詞語幹の並びになりますから、動詞語幹の最初の子音は脱落しません。

他動詞では唇音のbやmといった子音があり、調整された母音に強勢が落ちる場合、音位転換された動詞マーカー-u-や-o-がmの代わりに現れます。以下の通りの変化をたどります。

duálȩm 「それらを植える」の場合

mȩ + dálȩm + Ø （基本形） →
d + mȩ + álȩm + Ø （音位転換）→
d + m + álȩm + Ø （ȩの省略） →
d + u + álȩm + Ø （異化） →
d + u + álȩm （目的語代名詞のØの削除）

唇音のない場合はどうか見てみましょう。

dmásȩch　「それらを彫る」の場合

mȩ + dásȩch + Ø （基本形） →
d + mȩ + ásȩch + Ø （音位転換）→
d + m + ásȩch + Ø （ȩの省略） →
d + m + ásȩch （目的語代名詞のØの削除）

音位転換された動詞マーカーと母音の混交

音位転換された動詞マーカーは様々な音声的変化を伴います。その一つが母音の混交です。mȩlatȩch 「きれいにする」の三人称単数目的語現在完了形ではngotȩchii「それをきれいにした」ですが、この基本形はmȩ + ngatȩch + íi （すなわち、動詞マーカー＋動詞語幹＋目的語代名詞）です。

まず動詞マーカーの音韻転換が起き、動詞語幹のȩが削除されます。それでng + m + atȩch + íiとなります。動詞マーカーのmがuに変化し、ng + u + atȩch + íiになります。uとaが混交してoになります。結果、ngotȩchíiが出来上がります。まとめると以下のようになります。

mȩ + ngatȩch + íi （基本形） →
ng + mȩ + atȩch + íi （音位転換） →
ng + m + atȩch + íi （ȩの省略） →
ng + u + atȩch + íi （強勢の落ちない音節での動詞マーカーのuへの変化） →
ng + otȩch + íi （母音の混交）

高後母音のuの後と中前母音の中を保持した結果、中後母音のoができました。

mȩngȩlebȩd「ぶつ」の三人称単数目的語現在完了形cholȩbȩdiiのでき方

mȩ + chȩlebȩd+ íi（基本形） →
ch + mȩ + ȩlebȩd + íi （音位転換）→
ch + m + ȩlebȩd + íi（ȩの削除） →
ch + u + ȩlebȩd + íi（強勢の落ちない音節での動詞マーカーののuへの変化） →
ch + olebȩd + íi（母音の混交） →
ch + olȩbȩd + íi（強勢の落ちない音節でのeからȩへの変化）

mȩngȩlebȩd「ぶつ」の三人称複数非人間目的語現在完了形cholebȩdのでき方

mȩ + chȩlȩbȩd + Ø（基本形） →
ch + mȩ + ȩlébȩd + Ø（音位転換） →
ch + m + ȩlébȩd + Ø（ȩの脱落） →
ch + u + ȩlébȩd + Ø（強勢の落ちない音節での動詞マーカーのuへの変化） →
ch + olébȩd + Ø（母音の混交） →
ch + olébȩd （目的語代名詞でのØの脱落）

ここでも中後母音のoは高後母音のuの後と中前母音の中を保持しています。

結論として、oはu + a, u + e, u + ȩの結果、できるといえるでしょう。

音位転換した動詞マーカーの削除

以下の分布を見てみましょう。

l + uchȩs + íi（動詞マーカーの削除）

1 + il + ȩng + ír （強勢の落ちない音節でのeのȩへの変化）
